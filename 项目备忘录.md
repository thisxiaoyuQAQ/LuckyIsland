# 项目备忘录

## 项目概述
LuckyIsland —— Windows 灵动岛式桌面助手。常驻屏幕顶部中央的「单页画布」面板，聚合时间/日历/天气/股票/待办/终端，并通过 hook 接收 Claude/Codex 完成通知。

## 技术栈
- 框架：Tauri 2.x（Rust 后端 + WebView2 渲染）
- 前端：React 19 + TypeScript + Vite 6
- 样式：Tailwind CSS v4 + shadcn/ui
- 动画：Motion (Framer Motion)
- 状态：Zustand + persist
- 终端：xterm.js + portable-pty (Rust)
- 存储：SQLite (rusqlite) + TOML 配置
- 通知桥：lucky-notify CLI + 本地 HTTP
- AI 助手：AgentProvider 抽象（claude/codex CLI 优先）+ Porcupine 语音唤醒（方案见 docs/AI助手方案.md）

## 核心约定
- 配置：`%APPDATA%/LuckyIsland/config.toml`，`notify` crate 热重载
- 数据：SQLite `%APPDATA%/LuckyIsland/data.db`
- ID：UUID v4（待办、通知等）
- 时间：ISO 8601 字符串 + Unix timestamp 双存
- 错误：Rust `anyhow` + `thiserror`；前端 ErrorBoundary
- 日志：Rust `tracing` → 文件；前端 console + 上报
- **单页画布**：同一时刻只渲染一个页面，按键/页签切换
