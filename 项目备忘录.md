# 项目备忘录

## 项目概述
LuckyIsland —— Windows 灵动岛式桌面助手。常驻屏幕顶部中央的「单页画布」面板，聚合时间/日历/天气/股票/待办/终端，并通过 hook 接收 Claude/Codex 完成通知。

## 技术栈
- 框架：Tauri 2.x（Rust 后端 + WebView2 渲染）
- 前端：React 19 + TypeScript + Vite 7
- 样式：Tailwind CSS v4 + shadcn/ui
- 动画：Motion (Framer Motion)
- 状态：React 本地状态 + `useSyncExternalStore` 外部 store；跨窗口持久状态以 SQLite settings KV 为真源
- 终端：xterm.js + portable-pty (Rust)
- 存储：SQLite (rusqlite)，当前 schema 由 Rust 初始化；尚未建立版本化 migration
- 通知桥：lucky-notify CLI（本地 HTTP 客户端）+ 127.0.0.1:9753 HTTP 服务
- AI 助手：AgentProvider（Claude CLI / Codex CLI / 自定义 Chat API）+ 可取消单活请求；语音使用 sherpa-onnx + cpal，TTS 使用 Windows SAPI

## 核心约定
- 配置与数据：Tauri `app_data_dir()/data.db` 内的 SQLite；设置使用 `settings(key,value)` KV，不使用 config.toml 热重载
- ID：UUID v4（待办、通知等）
- 时间：ISO 8601 字符串 + Unix timestamp 双存
- 错误与日志：当前主要返回字符串错误并使用 `eprintln!`；尚未建立结构化错误、持久脱敏轮转日志
- **单页画布**：同一时刻只渲染一个页面，按键/页签切换
