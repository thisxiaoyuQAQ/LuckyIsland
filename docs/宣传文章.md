# LuckyIsland：我把 macOS 灵动岛搬到了 Windows，还顺手塞进了天气、股票、终端和一只电子木鱼

> 屏幕顶部一条透明小条，平时只显示时钟；需要时展开成一块全能画布。这是我在 Windows 上想要的「灵动岛」，现在它叫 LuckyIsland。

## 起因：任务栏太挤，桌面太乱

我用 Windows 干活，常年开着一堆东西：时钟、天气、待办、行情、终端，外加三四个 AI 助手窗口。任务栏挤成一团，桌面铺满小部件，想要的信息永远在最底层。

macOS 的灵动岛给了一个很好的启发——**信息不该是你要去找的，而该是你余光一扫就能看到的**。于是我想：能不能在 Windows 顶部放一条常驻的透明小条，平时安静地显示时间，需要的时候展开成一整块面板，把高频信息全收进去？

LuckyIsland 就是这个想法的答案。

## 它长什么样

平时，它是屏幕顶部中央一条 720 像素宽的透明小条，只显示一行时钟。毛玻璃背景，跟随系统深浅色，安静得像不存在。

按下 `Alt+X`，它向下展开成一块 720×400 的画布。你能在这块画布上秒切七个页面：**时间、日历、天气、股票、待办、终端、通知**。滚轮一滚就翻页，`Alt+1~9` 直接跳转。

再按 `Alt+Space`，一个独立的 AI 助手面板从顶部滑下来——Claude、Codex、还是你自己的 Chat API，三选一，随时问。

## 时间页：不是一张静态表盘，是一块能自己拼的画布

时间页是我最得意的部分。它不是传统那种「大字时钟 + 日期」就完事的页面，而是一块**可自定义的流式画布**：

- 中央是时钟，可以放在上方、中央、下方任意位置
- 时钟颜色支持纯色或双色渐变，内置 9 个主题预设（极光、日落、樱花、海洋、紫罗、森林、熔岩、星空、金辉）
- 周围环绕着五个可开关、可拖拽排序的小组件

这五个组件是：

**一言**——满行单行展开的一句随机文案，点击换一句。写代码卡住的时候瞥一眼，偶尔能撞上一句对路的。

**程序员历史上的今天**——「1991 · Python 发布」这样的条目，满行展示，可上下翻，点开看完整描述。每天换一批，当摸鱼读物正好。

**今日运势**——大吉 / 中吉 / 小吉 / 平 / 末小吉，配星级、幸运色、幸运数字。能反复重抽，直到抽到大吉为止（不是）。纯娱乐，但谁不想每天早上看点好话。

**电子木鱼**——这个一定要试。点一下，「咚」一声（用的是真实木鱼音效），功德 +1，木鱼还会放大再缩回。攒到 10、50、100 下有里程碑。**每到周四**，还会随机蹦出一句「疯狂星期四」文案，比如「木鱼一敲，烦恼清零。今天周四，V 我 50 看看实力。」今日功德和累计功德分开记，跨午夜自动清零今日、保留累计。

**今日心情**——五个表情选一个，记录连续打卡天数。漏记一天就断。简单，但能让你对自己最近的状态有个直观感觉。

这些组件想开哪个开哪个，想放哪放哪，想排什么顺序拖一下就行。每个人的时间页都可以长不一样。

## 不止时间页

**天气**接的是 uapis.cn，多城市、IP 自动定位、离线缓存、气象预警都有。断网时显示上次缓存的天气并标「离线」。

**股票**走腾讯行情 + sina 搜索，自选股拖拽排序，点开看日 / 周 / 月 K 线。红涨绿跌还是绿涨红跌随你选。

**日历**带农历、节气、节日，用 nongli 库算的。

**待办**是完整的 CRUD，带优先级和截止日期，SQLite 持久化。

**终端**是 xterm.js + portable-pty 的真终端，多 tab、快捷命令、一键打开外部 Windows Terminal。在灵动岛里敲命令，比开一个完整终端窗口轻量得多。

**通知页**背后是一个本地 HTTP 端点 `127.0.0.1:9753/notify`，任何脚本都能往里 POST 一条通知，LuckyIsland 会存历史、弹 Windows toast、并把灵动岛展开跳到通知页。

## 最香的是：能接 AI 的完成通知

如果你也用 Claude Code 或 Codex CLI 写代码，你一定有过这种体验：让 AI 跑一个长任务，然后你切去干别的，二十分钟后想起来——哎，跑完没？

LuckyIsland 自带 `lucky-notify` CLI 和 Claude / Codex 的 hook 配置。配好之后，AI 每次完成任务，会自动往灵动岛发一条通知，Windows toast 弹出来，你不用一直盯着终端。

```bash
curl -X POST http://127.0.0.1:9753/notify \
  -H "Authorization: Bearer <token>" \
  -d '{"title":"Claude 完成","body":"重构了 time 模块","source":"claude"}'
```

这个端点是通用的——CI 构建、定时任务、任何脚本都能调。LuckyIsland 既是桌面助手，也是你的「通知中心」。

## 还能语音唤醒

如果你愿意，可以开语音：喊一声唤醒词，它应一句「主人我在」，然后听你说话、转写、交给 AI 回答、再用 Windows 自带的 SAPI5 语音念出来。全程 sherpa-onnx 做唤醒和 ASR，模型唤醒时才懒加载，闲置只占 ~110MB 内存。

不想开语音？关掉就是了，基线 61MB，CPU 不到 1%，装上常驻毫无负担。

## 性能与体积

做桌面常驻工具，最怕又重又大。LuckyIsland 的数字：

- 安装包 **6.3MB**（Tauri + thin LTO + strip）
- 语音关闭基线 **~61MB** 内存，CPU **<1%**
- 语音开启闲置 **~110MB**

整个后端是 Rust，前端 React 19，存储 rusqlite，网络 reqwest。没有 Electron 那套 Chromium 包袱，5MB 出头的安装包就能跑起来。

## 多屏与外观

多屏用户没被遗忘：设置里能选灵动岛挂在主屏还是某个副屏，选完立即移动、重启恢复。运行时副屏断开会自动回退主屏并保留原选择；副屏恢复后应用不主动跳回，可重启或手动重选。部分 Windows 环境可能自行恢复窗口位置，但这不是应用承诺。

外观上，窗口透明度 10%~100% 可调，横向纵向偏移实时预览，能把它从顶部中央挪到任何你习惯的位置。整个配置支持导入导出，但密钥、token、缓存这些敏感 / 运行数据不会跟着导出——换机器时只搬配置，不搬隐私。

## 怎么用

```bash
pnpm install
pnpm tauri dev    # 开发
pnpm tauri build  # 打包成 NSIS 安装包
```

装好之后，`Alt+X` 显示灵动岛，`Alt+Space` 唤起 AI，滚轮切页面。进设置（托盘右键 -> 设置...）慢慢调你的时间页组件和主题。

## 写在最后

LuckyIsland 不是要替代任何专业工具——它不替代你的 IDE、你的行情软件、你的待办 App。它想做的是**那一块你不需要切换窗口就能瞥见的屏幕角落**。

时钟、一句话、今天的历史、今天的运势、今天的功德、今天的心情——这些碎片化的、低频但让人安心的小信息，本来就该有一个轻量的地方安放。再加上天气、行情、待办、终端、通知、AI、语音这些真正干活的页面，它就成了一个你愿意常驻的东西。

如果你也常驻 Windows 桌面，不妨给它一个位置。
