# RUN-10C-05 24h 日常功能长跑

> 来源：AUD-RUNTIME-05（与 vault/07-DIY打磨 P2 既有项共用）
> 通过条件：无崩溃和持续无界增长；阈值先由用户确认

## 前置阈值（执行前请确认 / 调整）

- 长跑时长：≥ 24h（允许分段累计）
- 采样间隔：≤ 1h
- 内存（工作集）增长：24h 末次 ≤ 首次的 1.5×（语音未持续开启场景）
- CPU 空闲平均：≤ 3%
- 句柄数增长：≤ 首次的 2×
- 崩溃 / 无响应：0 次
