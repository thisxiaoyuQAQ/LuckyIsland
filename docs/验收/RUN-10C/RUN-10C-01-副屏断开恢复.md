# RUN-10C-01 副屏断开 / 恢复（多 DPI / 显卡）

> 来源：AUD-RUNTIME-01
> 通过条件：断开主动回退、恢复主动移回已保存副屏；偏移/状态保留且不无故抢焦点

## 前置

- 主屏 + ≥1 副屏（推荐两屏 DPI 不同，如 100% + 150%）
- 已在「设置 → 通用 → 显示器」选择副屏并保存
- 已设置窗口偏移（X/Y 均非 0）
- 启动 `pnpm tauri dev` 或 Release 安装包

## 步骤与预期

| # | 操作 | 预期 |
|---|---|---|
| 1 | 在副屏正常显示灵动岛，记录坐标与偏移 | 岛位于副屏，坐标 = 选中副屏基线 + 用户偏移 |
| 2 | 物理断开副屏（拔 HDMI / Win+P 选「仅电脑屏幕」） | ≤ 2s 内岛自动回退到主屏，不卡死、不丢焦点；坐标为主屏基线 + 同样的用户偏移 |
| 3 | 在回退状态下切页（天气/股票/终端） + 展开/收起 | 页面切换与展开动画正常，无错位 |
| 4 | 重新接回副屏 | ≤ 2s 内岛自动移回已保存副屏；坐标恢复为副屏基线 + 用户偏移 |
| 5 | 步骤 2→4 循环 3 次 | 每次都稳定回退/恢复，无累积漂移 |
| 6 | 不同 DPI 副屏（若可）重复步骤 1–5 | 坐标按目标副屏 scale factor 正确换算 |
