# RUN-10C-03 整窗手动穿透开启 / 关闭与恢复入口

> 来源：AUD-RUNTIME-03
> 前置决策：PROD-10C-05（Hidden 与整窗穿透分离，穿透为手动开关 + 可选热键）
> 通过条件：开启时整窗点击落到后方；关闭后岛可交互；设置页 / 热键可恢复

## 前置

- 当前 build 含整窗穿透实现（模块 11）
- 在「设置 → 通用 → 整窗穿透」可看到开关；热键面板可绑定「切换整窗穿透」
- 桌面有可被点击验证的后方窗口（如浏览器 / 资源管理器）

## 步骤与预期

| # | 操作 | 预期 |
|---|---|---|
| 1 | 默认状态：穿透关闭 | 岛整体可点击（顶栏、切页按钮、输入框均响应） |
| 2 | 设置面板打开穿透 | 岛立即可见但鼠标事件全部穿透到后方窗口；点击岛中任意位置都落在后方窗口上（可用浏览器链接是否被点中验证） |
| 3 | 通过设置面板再次关闭穿透 | 岛恢复可交互 |
| 4 | 在热键面板给「切换整窗穿透」绑定一个键（如 Ctrl+Alt+P） | 绑定成功；按一下开启、再按一下关闭，行为同步骤 2/3 |
| 5 | 穿透开启状态下使用「隐藏岛」快捷键（Alt+X 默认） | 岛完全隐藏（不再是「可见但穿透」） |
| 6 | 再次按 Alt+X 唤醒岛 | 岛重新显示；穿透状态保留（仍是穿透） |
| 7 | 穿透开启 + 隐藏 + 唤醒 循环 3 次 | 状态一致不漂移 |
| 8 | 穿透开启 + 全屏应用（如视频播放器全屏） | 岛按 hide_in_fullscreen 规则隐藏；退出全屏后恢复 |
