## 接口与类型
```ts
type PageId = "time" | "calendar" | "weather" | "stock" | "todo" | "terminal" | "notify";
interface PageConfig { id: PageId; enabled: boolean; }
```
Rust 命令：
- `todo_create(input)`, `todo_list()`, `todo_update(id, patch)`, `todo_delete(id)`
事件：`todo://changed`

## 实现要点
- 页面切换器：页签按钮 + Alt+1~9 + Alt+←/→ + Tab；当前页指示器
- Motion `AnimatePresence` 做切换动画（morph/滑入，≤280ms）
- 页面注册表：从 config 读 enabled + order，惰性挂载（终端页尤其需要）
- 时间页：HH:MM 大字 + 可选秒/日期/星期；每秒 tick
- 日历页：月视图 + `lunisolar` crate 算农历/节气 + 今日待办数角标
- 待办页：列表 + 内联添加 + 完成/删除/编辑；按优先级/截止排序
- 待办持久化：SQLite `todos` 表（id uuid, title, done, priority, due_at, created_at）

## 测试要点
- Alt+1~9 跳转正确页面
- 页签点击切换
- 切换动画流畅
- 待办增删改查 + 重启后数据保留
- 农历显示正确（用已知日期校验）
