## 接口与类型
Rust 命令：
- `term_create(cwd, shell?) -> term_id`
- `term_write(term_id, data)`, `term_resize(term_id, cols, rows)`, `term_kill(term_id)`
事件：`term://output { term_id, data }`

## 实现要点
- 前端 xterm.js + addon-fit + addon-webgl + addon-web-links
- 后端 `portable-pty` 创建 PTY，tokio task 读输出推事件
- 默认 shell：PowerShell 7 → Windows PowerShell → cmd
- 多 tab：每 tab 一个 PtyPair，关闭时 kill 子进程
- 快捷命令：config `[[terminal.shortcuts]]`，一键在新 tab 执行
- 「外部 wt 打开」按钮：spawn `wt.exe -d <cwd>`
- 大输出：webgl 渲染 + scrollback 限制

## 测试要点
- 终端可输入输出（ls/echo）
- 多 tab 新建/关闭无残留进程
- resize 同步
- 快捷命令执行（如 codex/git pull）
- 大输出（`dir /s`）不卡顿
