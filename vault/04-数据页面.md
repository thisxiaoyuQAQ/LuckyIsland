## 接口与类型
Rust 命令：
- `weather_get(city) -> WeatherNow`
- `stock_get(symbols) -> Vec<Quote>`
- `stock_watchlist_add(sym)`, `stock_watchlist_remove(sym)`
事件：`stock://tick { quotes: [...] }`
配置：`[weather] [stock]`

## 实现要点
- 天气：和风天气 QWeather API（key 从 env），10min 刷新，离线显示最近一次
- 股票：腾讯 `qt.gtimg.cn` 接口，交易时段 3–5s 轮询，非交易段 30s
- HTTP 走 Rust `reqwest`，带 Referer，避免前端 CORS 与密钥泄露
- 失败重试 + 指数退避 + 缓存最近一次到 SQLite
- 涨跌着色（红涨绿跌可配置）
- 紧凑态轮播自选股

## 测试要点
- 天气能拉到并显示
- 股票实时刷新（交易时段）
- 断网时显示缓存 + 离线标识
- 自选股增删持久化
