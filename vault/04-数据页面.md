## 接口与类型
Rust 命令：
- `weather_get(city?) -> WeatherNow`（city 可选，缺省读 settings `weather:city`，再缺省 "北京"；失败回退缓存且 `offline=true`）
- `weather_get_city() -> String`、`weather_set_city(city)`
- `stock_get() -> Vec<Quote>`（一次性拉取，失败回退缓存）
- `stock_watchlist_list() -> Vec<WatchItem>`、`stock_watchlist_add(sym)`、`stock_watchlist_remove(sym)`
事件：`stock://tick`，payload 为 `Vec<Quote>`（裸数组，非 `{quotes:[...]}`）
配置：M3 不引入 config.toml，用 SQLite settings KV（城市/缓存）+ env + 常量（轮询间隔）；config.toml 留到 M6

## 实现要点
- 天气：**uapis.cn** `https://uapis.cn/api/v1/misc/weather?city=`（免 key，UTF-8 JSON），10min 刷新由前端按需触发，离线显示最近一次缓存
  - 仅返回当前天气+预警；7天/逐小时预报是 uapis 另接口，留后续
  - icon 码为和风风格（100 晴 / 101 多云 / 104 阴 / 30x 雨 / 40x 雪 / 50x 雾霾）
- 股票：腾讯 `qt.gtimg.cn`，**GBK 编码**（encoding_rs 解码），交易时段 5s / 非交易 30s 轮询
  - 字段下标：name=1 code=2 current=3 yesterday=4 open=5 high=33 low=34 change=31 change%=32 time=30
  - 代码规整：前缀小写（sh/sz/us/hk/bj），代码保留大小写（usAAPL）
- HTTP 走 Rust `reqwest`（共享 Client），带 Referer，避免前端 CORS 与密钥泄露
- 失败回退 SQLite 缓存（settings KV `weather:last` / `stock:last`）
- 涨跌着色：红涨绿跌（硬编码，M6 可配置）
- 紧凑态：天气 图标+温度+天气；股票 首条+计数
